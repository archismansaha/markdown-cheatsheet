#Welcome , Here is a cheatsheet for markdown language.

Heading:

 syntax: #    heading 1 
 syntax: ##   heading 2 
 syntax: ###  heading 3 
 syntax: #### heading 4

Output:

heading1

heading2

heading3

heading4

bold and italic

syntax: **text to bold**

syntax: *text to italic*

Output:

text to bold

text to italic

task list:


syntax: - [x] task 1
        - [ ] task 2

Output:

  • [x] task 1
  • [ ] task 2

Ordered and unordered lists:

syntax:  - text to be in unorederd list
syntax:  1.  text to be in ordered list
            2. text to be in ordered lists.

You can make nested ordered and unordered list:
- text 1
    - text 2

1. text 1 
    2.text 2 

2. text 1 
     - text 2

Output:

  • text to be in unorederd list
  • text to be in ordered list
  • text to be in ordered lists.
  • text 1
    • text 2

  • text 1

    1. text 2

  • text 1

    • text 2

Blockqoutes:

> this is a single block quote.
>> nested block quote.

Output:

this is a single block quote.

nested block quote.

Code Blocks :

synatx: ``` code

(write code in next line) print("Have a great day")

***
## Output:
***

print("Have a great day")
```
